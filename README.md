# Instagram Clone - iOS 📸

A feature-rich iOS application that replicates the core functionalities of Instagram. Built completely programmatically and with Storyboards using Swift, UIKit, and Firebase.

## ✨ Features

*   **User Authentication:** Secure Login and Logout functionality using Firebase Auth.
*   **Real-time Home Feed:** View a timeline of posts fetched dynamically from Firestore.
*   **Interactive Engagement:** 
    *   Like and unlike posts with real-time UI updates (Heart animation).
    *   View real-time like counts.
*   **Dynamic Commenting System:**
    *   Add comments to any post.
    *   View a list of comments with auto-expanding (dynamic height) cells.
*   **Side Menu Navigation:** Smooth side menu for accessing settings and user profile.
*   **Modern UI:** Clean, responsive, and user-friendly interface built with Auto Layout.

## 🛠 Technologies & Tools

*   **Language:** Swift 5
*   **UI Framework:** UIKit (Storyboards & Auto Layout)
*   **Backend:** Firebase
    *   `FirebaseAuth` (User Authentication)
    *   `FirebaseFirestore` (NoSQL Database for Posts, Comments, and Likes)
*   **Architecture:** MVC (Model-View-Controller)
*   **Dependency Manager:** Swift Package Manager (SPM) / CocoaPods
